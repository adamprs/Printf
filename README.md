# ft_printf

## Description

ft_printf consists of reimplementing the standard C `printf()` function.

The goal of this project is to learn how to handle variadic functions and reproduce formatted output behavior. It involves parsing a format string and printing different types of data accordingly.

The implementation supports basic conversions such as characters, strings, pointers, integers, unsigned numbers, hexadecimal formats, and the percent symbol, while mimicking the behavior of the original `printf`.
